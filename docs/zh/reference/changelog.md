# 更新日志

本文档记录 AnyParse 的发布历史，包括主要功能变更、兼容性更新以及重要修复。

---

## 1.1 系列版本

### 1.1.2 (2026/07/01)

- PDF 文档：新增 .pdf 格式支持
- Office 文档：新增 .docx, .pptx, .xlsx 格式支持
- 文本文档：新增 .txt, .md, .rst, .html, .xhtml, .htm, .shtml 格式支持
- 电子表格：新增 .csv, .tsv 格式支持
- 图像：新增 .png, .jpg, .jpeg 格式支持
- 电子书：新增 .epub 格式支持
- Jupyter 笔记本：新增 .ipynb 格式支持
- 其他：新增 .eml（电子邮件）格式支持
