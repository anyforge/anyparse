# Changelog

This document records the release history of AnyParse, including major capability changes, compatibility updates, and notable fixes.

---

## 1.1 Series Versions

### 1.1.1 (2026/07/01)

- pdf documents: add support for .pdf
- office documents: add support for .docx, .pptx, .xlsx
- text documents: add support for .txt, .md, .rst, .html, .xhtml, .htm, .shtml
- spreadsheets: add support for .csv, .tsv
- images: add support for .png, .jpg, .jpeg
- epubooks: add support for .epub
- jupyter notebooks: add support for .ipynb
- others: add support for .eml (email)
